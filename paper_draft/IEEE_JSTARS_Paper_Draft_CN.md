## 摘要

基于深度学习的滑坡监测在遥感影像自动制图中表现出良好潜力，但其实际部署仍受到像素级标注成本高昂这一关键瓶颈的制约。在滑坡场景中，标注不仅需要地质领域专家知识，还需要对滑坡边界进行细致描绘，因此代价尤其高。主动学习（Active Learning, AL）通过优先选择最具信息量的样本进行标注，为缓解该问题提供了自然思路。然而，现有面向语义分割的主动学习策略通常依赖单一准则，如不确定性或多样性，因此难以在不同学习阶段自适应平衡探索与利用；同时，它们对样本被查询的原因缺乏足够可解释性。为解决上述问题，本文提出 AAL-SD，一种面向滑坡语义分割的 LLM Agent 驱动自适应主动学习框架。该框架的核心是 AD-KUCS（Adaptive Dynamic Knowledge-Uncertainty Sampling）算法，通过动态调整的权重因子融合不确定性与知识增益。在本文报告的 full-model 设置中，查询权重由 warmup 与风险感知的闭环策略生成，而 LLM agent 负责基于上下文检查候选样本，并以可解释理由最终确定查询子集。在 Landslide4Sense 基准上的实验表明，在代表性的 seed-42 实验中，策略控制版 AAL-SD 的最终 mIoU 达到 0.7651、最终 F1-score 达到 0.8490、ALC 达到 0.6693。进一步的四 seed 分析表明，没有任何单一方法在所有指标上都占优：Wang-style 获得最高平均 ALC（0.6682 +/- 0.0037），策略控制版 AAL-SD 紧随其后（0.6674 +/- 0.0029），而 agent-lambda 版本获得最高平均最终 mIoU（0.7616 +/- 0.0014）。这些结果说明，AAL-SD 的主要价值不在于宣称对所有基线形成全面压制，而在于提供一个具有竞争力、可解释、可审计的遥感主动学习控制框架。

**索引词：** 主动学习，滑坡检测，语义分割，LLM agent，遥感，深度学习。

## I. 引言

### A. 研究背景与动机

滑坡是全球范围内破坏性最强的地质灾害之一，每年都会造成严重的人员伤亡、财产损失和经济损害。随着遥感技术的发展以及高分辨率卫星影像的日益丰富，基于遥感影像的自动滑坡检测已成为灾害减灾与预警系统中的关键工具 [1]。深度学习方法，尤其是 U-Net、DeepLabV3+ 等语义分割模型，在光学遥感影像滑坡边界提取任务中取得了显著进展 [2]-[4]。这类数据驱动方法能够直接从原始影像中学习层次化特征，无需人工特征工程，相较传统机器学习方法表现出更强的精度与泛化能力。

然而，深度学习模型优异性能的背后是对大规模标注数据的高度依赖。在滑坡检测场景中，获取像素级标注面临多重挑战。首先，滑坡标注需要专业地质知识支持，导致标注过程昂贵且耗时。其次，滑坡事件在空间与时间上均具有高度异质性，不同地区的地貌形态差异显著，因此需要多样化训练样本。第三，滑坡数据通常存在严重类别不平衡问题，滑坡像素往往只占整幅图像的很小比例。这些因素共同构成了深度学习滑坡检测系统落地应用的主要瓶颈。

### B. 主动学习作为解决方案

主动学习（AL）是缓解机器学习标注瓶颈的重要范式 [5]。其核心假设在于：若学习算法能够主动选择最具信息量的样本进行标注，则在相同标注预算下，可较被动随机采样获得更优性能。通过从未标注池中迭代选择高价值样本，并向标注者（如人工专家）查询其标签，主动学习旨在最大化有限预算下的学习效率。

在遥感语义分割任务中，主动学习尤具价值，因为像素级标注相较图像级分类标注要耗费更多人力。滑坡标注通常要求专家精确勾勒滑坡边界，一张图像可能需要数小时才能完成标注。因此，主动学习在降低标注负担的同时保持检测精度方面具有明显潜力。

现有语义分割主动学习策略大致可分为三类：基于不确定性的策略、基于多样性的策略，以及混合策略。基于不确定性的策略（如熵采样、间隔采样）选择模型预测最不确定的样本，以细化判别边界 [6]；基于多样性的策略（如 Core-Set）选择能够最大化特征空间覆盖度的样本，以促进探索 [7]；混合策略则尝试同时结合这两类信息，但往往采用固定权重或人工调参。

尽管已有研究取得一定进展，现有策略在实际滑坡检测任务中仍存在明显局限。第一，单一指标方法无法随学习进程动态调整：早期阶段通常更需要利用不确定性优化判别边界，后期阶段则更需要扩大样本覆盖以补充知识盲区。第二，这些方法通常缺乏可解释性，样本为何被选中的原因对人工使用者并不透明。第三，大多数方法依赖固定权重，无法根据模型状态反馈自适应调节探索与利用之间的平衡。

### C. LLM Agent：一种新范式

近年来，大语言模型（Large Language Models, LLMs）尤其是具备 Chain-of-Thought（CoT）推理能力的模型，为机器学习系统中的智能决策带来了新可能 [8]-[9]。不同于传统算法式方法，LLM agent 能够进行复杂推理、理解上下文信息，并生成人类可理解的决策解释。ReAct 框架表明，LLM 可以通过“推理-行动”交替方式，将思维过程与工具调用、环境观测有机结合起来 [10]。

受此启发，本文尝试将 LLM agent 引入滑坡语义分割的主动学习决策闭环。该 agent 可接收当前学习状态的综合信息，例如不确定性得分、知识增益指标和训练进展，并通过自然语言推理生成更具上下文感知的样本选择决策。这一思路推动主动学习从“纯计算驱动”走向“认知-计算协同”，使样本选择更加灵活且可解释。

### D. 主要贡献

本文的主要贡献可概括为以下五点：

1. **提出新框架**：本文提出 AAL-SD，将 LLM agent 以可审计的工具调用方式纳入遥感滑坡语义分割主动学习决策闭环，用于基于上下文的候选样本检查、最终选择与可解释输出，并与数值闭环策略层协同实现“认知-计算”混合控制。

2. **提出自适应算法**：本文设计 AD-KUCS（Adaptive Dynamic Knowledge-Uncertainty Sampling）动态混合查询策略，将不确定性与知识增益融合在统一打分函数中。基础形式支持基于进程的自适应权重，而本文报告的 full-model 进一步采用 warmup 与风险感知的闭环 lambda 策略，以在过拟合与回滚信号下保持稳定选样。

3. **提出闭环优化视角**：本文将 AAL-SD 解释为闭环优化控制器，而不仅仅是静态排序规则。框架利用梯度导出的风险信号调节查询权重和所选样本，从而间接改变标注数据分布，并进一步影响后续训练轮次的梯度轨迹。

4. **提升可解释性**：LLM agent 可为每轮样本选择生成自然语言解释，为主动学习过程提供前所未有的透明性，这对于构建自动化标注系统的信任尤为重要。

5. **系统性实验验证**：本文在 Landslide4Sense 基准数据集上进行了大量实验，并补充了刷新后的四 seed 比较，结果表明 AAL-SD 在强不确定性、多样性与混合基线面前保持高度竞争力，同时提供更强的控制可解释性。

本文其余部分安排如下：第二节回顾语义分割主动学习、滑坡检测深度学习方法以及 LLM 在机器学习中的应用；第三节介绍本文方法，包括 AD-KUCS 算法与 LLM agent 决策系统；第四节给出实验设置；第五节展示实验结果并进行分析；第六节讨论方法的意义、局限性与未来工作；第七节给出全文结论。

## II. 相关工作

### A. 面向语义分割的主动学习

近年来，语义分割主动学习受到广泛关注，其目标是设计高效查询策略，从未标注样本中识别最值得标注的图像块或样本。现有方法大体可以分为三类。

**基于不确定性的方法：** 这类方法选择模型预测最不确定的样本。熵采样通过计算整幅图像的平均像素熵来选择高不确定性样本 [6]。基于 Monte Carlo（MC）Dropout 的方法通过多次随机前向传播预测方差估计不确定性 [11]。基于置信差的方法则选择前两类预测概率最接近的样本 [12]。已有研究表明，在遥感分割任务中，不确定性采样能够以远低于全监督的标注成本获得接近的性能 [13]。

**基于多样性的方法：** 该类方法旨在最大化选中样本之间的差异性。Core-Set 通过在特征空间中最大化覆盖度来选择最具代表性的样本 [7]。密度加权方法同时考虑不确定性和样本在特征空间中的密度，从而避免冗余样本被重复选择 [14]。ViewAL 在三维场景理解中引入视角熵，表明融合视角多样性有助于提升主动学习效果 [15]。

**混合方法：** 将不确定性与多样性结合通常能获得更好性能。DIAL（Deep Interactive and Active Learning）提出了面向遥感语义分割的深度交互式主动学习框架，将用户交互与主动学习策略结合起来，引导用户优先标注最有价值区域 [16]。但此类方法通常采用固定权重，缺乏随学习进展自适应调节的能力。

### B. 基于深度学习的滑坡检测

深度学习方法显著推动了遥感滑坡检测的发展。U-Net、SegNet 和 DeepLabV3+ 等语义分割模型已成为像素级滑坡制图的主流方法 [2]-[4]。其中，DeepLabV3+ 通过空洞可分离卷积与编码器-解码器结构，在捕获多尺度上下文信息和滑坡边界细节方面具有突出优势 [4]。

Ghorbanzadeh 等人提出的 Landslide4Sense 数据集 [17] 为滑坡检测研究提供了标准化基准。该数据集包含多光谱 Sentinel-2 遥感影像及像素级标注，为不同方法之间的公平对比提供了基础。近期研究还从注意力机制 [18]、迁移学习 [19] 和多模态融合 [20] 等方面进一步提升滑坡检测性能。

不过，这些方法的共同问题是都需要大量标注数据。滑坡影像标注需要专业地质知识和大量人工投入，而类别不平衡问题又进一步放大了标注难度。因此，主动学习成为减少滑坡检测标注依赖、同时维持性能的重要方向。

### C. 机器学习中的大语言模型

大语言模型的发展已显著改变了机器学习多个子领域。GPT-4、Claude 等模型展现出强大的自然语言理解、推理和代码生成能力 [8]。Chain-of-Thought 提示技术则能够显式生成中间推理步骤，从而显著提升复杂任务表现 [9]。

进一步地，LLM 作为 agent 融入机器学习流程的趋势也日益明显。ReAct 框架证明，LLM 可以在工具调用与环境观测的基础上进行连续推理与行动 [10]，从而具备状态感知、信息更新和未来规划能力。

在数据标注与主动学习方向，已有工作开始探索使用 LLM 作为标注者或辅助标注者 [21]。例如，MEGAnno+ 提出了人机协同标注范式，使 LLM 先生成标签，再由人工结合主动学习策略进行验证 [22]。但这些工作主要聚焦于文本分类或图像分类，针对语义分割和遥感应用的探索仍然较少。

本文在此基础上进一步推进，将 LLM agent 直接纳入语义分割主动学习决策闭环，实现由上下文推理驱动的动态、可解释样本选择。

## III. 方法

### A. 问题定义

本文将滑坡语义分割主动学习问题形式化如下：给定未标注样本池 $U = \{x_1, x_2, ..., x_n\}$、一个较小的初始标注集 $L_0$ 以及固定测试集 $T$，目标是在总标注预算 $B$ 约束下，迭代地从 $U$ 中选择样本进行标注，使得在不断扩充的标注集 $L$ 上训练得到的模型在测试集 $T$ 上获得尽可能高的性能。

在每一轮主动学习迭代 $t$ 中，我们使用当前标注集 $L_t$ 训练语义分割模型 $f_\theta$。随后，模型对未标注池 $U$ 中所有样本进行预测，并通过采样函数 $a(x, f_\theta)$ 为每个未标注样本分配得分。得分最高的前 $k$ 个样本将被送往人工标注，然后加入 $L_{t+1}$，这一过程持续进行直至预算耗尽。

本文的一个重要视角是：采样决策不仅影响标注效率，也会改变下一轮优化问题本身。当新的查询样本被加入 $L_t$ 后，经验训练分布将发生变化，后续目标函数 $\mathcal{L}_{t+1}(\theta)$ 也随之在新的标注池上进行优化。因此，自适应查询控制虽然并不直接修改优化器内部的梯度计算公式，但会通过改变进入标注池的样本集合，间接影响后续训练轮次的梯度轨迹 $\nabla_\theta \mathcal{L}_{t+1}(\theta)$。

图 1 展示了 AAL-SD 的双线架构。该框架围绕两条在每轮主动学习中并行运行的交互循环组织：

**主动学习线**（图 1 上方，蓝色系）。该线路承载数据流：(1) 初始化数据池（$L_0$、$U$、$T$）；(2) DeepLabV3+ 分割模型在当前标注集 $L_t$ 上训练（每轮 10 个 epoch）；(3) 训练后的模型对未标注池进行推理，产生像素级概率图 $P(x)$ 与深层特征表示 $f(x)$；(4) AD-KUCS 计算不确定性 $U(x)$ 与知识增益 $K(x)$，将两者归一化至 $[0,1]$ 并融合为 $\text{Score}(x) = (1-\lambda_t) U(x) + \lambda_t K(x)$；(5) LLM agent 通过 ReAct 循环（get\_system\_status → get\_top\_k\_samples → finalize\_selection）检查候选样本，并以可解释理由提交查询子集 $Q_t$；(6) 被查询样本从 $U$ 移入 $L_{t+1}$，完成本轮。

**学习控制线**（图 1 下方，红色系）。该线路承载控制流，用于调节采样行为：(1) 训练过程中，逐 epoch 计算训练批次与测试批次探针之间的梯度对齐度（train-validation gradient cosine）；(2) 这些统计量被汇总为过拟合风险评分与回滚标志；(3) 风险信号输入 $\lambda$ 策略引擎，该引擎实现三阶段 warmup + 风险感知闭环调度（第 1-2 轮：$\lambda=0$，纯不确定性；第 3 轮：$\lambda=0.2$，warmup；第 4 轮起：带 EMA 平滑与 clamp 的风险响应闭环）；(4) 策略确定的 $\lambda_t$ 同时注入 AD-KUCS 打分公式与 agent 的系统状态上下文。

两条线路在 AD-KUCS 打分模块与 LLM agent 模块处汇聚（图 1 中以黄色虚线框标注为"汇聚区"）。这一双线设计是本文的核心架构贡献：学习控制线确保采样决策不是由固定启发式驱动，而是基于优化诊断信号持续适应；主动学习线则确保 agent 的可解释推理在控制策略设定的边界内运作。需要强调的是，AD-KUCS 是主动学习闭环内部的查询算法，而不是分割网络本身；真正执行训练与推理的是语义分割模型，AD-KUCS 仅利用其概率图与特征表示对未标注样本进行排序。

### B. AD-KUCS 算法

本文方法的核心是 AD-KUCS（Adaptive Dynamic Knowledge-Uncertainty Sampling）算法。该算法通过动态调整权重，将不确定性与知识增益两类信息融合在统一评分函数中。

**不确定性度量 $U(x)$：** 本文采用基于熵的不确定性估计。对于样本 $x$，设其包含 $N$ 个像素，模型对每个像素 $i$ 输出类别概率分布 $p_i$。则样本级不确定性定义为平均像素熵：

$$U(x) = -\frac{1}{N} \sum_{i=1}^{N} \sum_{c=1}^{C} p_i(c) \log p_i(c)$$

其中，$C$ 为类别数。熵越高，表示模型对该样本像素分类越不确定，因此该样本越值得被标注。

**知识增益度量 $K(x)$：** 我们采用受 Core-Set 启发的方法衡量样本的新颖性或覆盖缺口。对每个未标注样本 $x$，利用分割模型倒数第二层提取其特征表示 $f_x$。知识增益定义为：

$$K(x) = \frac{\min_{l \in L} \|f_x - f_l\|}{D_{max}}$$

其中，$D_{max}$ 表示当前标注集 $L$ 中样本两两间的最大距离，用于归一化。$K(x)$ 越高，说明该样本位于特征空间中相对欠覆盖的区域，可能带来更多新知识。

**动态权重 $\lambda_t$：** AD-KUCS 的关键创新在于自适应调整权重参数 $\lambda_t$。在采样器层面，其默认先验可写为基于标注进度的 Sigmoid 函数：

$$\lambda_t = \frac{1}{1 + e^{-\alpha \cdot (t/T_{max} - 0.5)}}$$

其中，$t$ 表示当前标注进度，$T_{max}$ 表示总预算，$\alpha$ 为控制曲线陡峭程度的参数。在学习早期，$\lambda_t$ 保持较低水平，更偏向不确定性采样以细化决策边界；随着训练推进，先验会逐渐增强对知识增益的关注。在本文报告的 full-model 配置中，这一先验进一步被 warmup 与风险感知的闭环策略修正：当出现过拟合预警或回滚相关信号时，系统会压低或限制 $\lambda_t$。这一机制并不直接改变优化器内部的梯度计算，而是通过控制进入标注池的样本集合，间接塑造后续优化行为。

**最终采样函数：** 综合后，每个样本的最终得分定义为：

$$Score(x) = (1 - \lambda_t) \cdot U(x) + \lambda_t \cdot K(x)$$

该形式使算法能够在整个学习过程中自适应平衡探索与利用。

### C. LLM Agent 决策系统

在 AAL-SD 框架中，LLM agent 作为智能控制器，其功能超越了单纯的分数排序。

**Agent 架构：** agent 接收多源上下文信息，包括：(1) 当前学习统计量，如轮次、模型性能和预算状态；(2) 候选样本信息，如前 $k$ 个样本的 $U(x)$、$K(x)$ 和 $Score(x)$ 值；(3) 历史选择记录；(4) 系统约束，如回滚阈值与过拟合风险指标。

**提示模板设计：** 本文设计了结构化 prompt 模板，引导 agent 进行 Chain-of-Thought 推理。该模板包含：

- 任务描述：“你是用于滑坡语义分割的主动学习控制器……”
- 当前状态：轮次、剩余预算、近期性能变化
- 可用动作：查询系统状态、检查候选样本，并提交最终选择；在专门消融设置中，也可以开放显式 lambda 控制
- 输出格式：便于解析的结构化 JSON

**工具箱接口：** agent 可调用如下工具：

- `get_system_status()`：返回当前学习状态
- `get_score_distribution(...)`：返回不确定性与知识增益的统计分布
- `get_top_k_samples(k, lambda_param)`：返回前 k 个候选样本
- `get_sample_details(sample_id)`：返回指定样本详细信息
- `finalize_selection(sample_ids, reason)`：提交最终选择并附解释

对于本文主结果所使用的 full-model 配置，agent 不允许直接调整 query size、epochs 或 alpha，lambda 也由闭环策略层提供，而不是由 agent 逐轮显式控制。显式 `set_lambda(...)` 仅在独立的 `full_model_B_lambda_agent` 变体中开放。

**决策流程：** agent 按如下步骤进行推理：

1. 分析当前学习阶段与模型状态
2. 查看候选样本及其评分指标
3. 参考历史选择模式
4. 应用启发式规则（如避免连续从同一区域选择样本）
5. 输出最终选择结果及自然语言解释

这一过程使样本选择具备更强可解释性，能够说明某一轮为何基于不确定性、多样性或策略考虑选择特定样本。

### D. 风险控制与恢复信号

为确保学习过程稳定，本文引入了风险控制与恢复感知监控机制：

**过拟合检测：** 系统使用基于梯度对齐的训练-验证差异指标，尤其是 train-validation gradient cosine 统计量，监控潜在过拟合风险；一旦偏差超过阈值，查询策略将转入更保守的控制模式。

**基于梯度信号的自适应控制：** 框架利用梯度导出的风险信号以及近期性能变化共同调节 $\lambda_t$。当梯度失配表明优化不稳定时，控制器会偏向更保守、更安全的选样方式，从而形成“优化诊断 -> 数据采集”的闭环。

**自适应阈值监控：** 系统使用由近期指标波动导出的自适应阈值追踪性能趋势。若性能出现显著下滑，则激活 rollback flag 与保守控制信号。

**基于检查点的恢复支持：** 框架在每一轮后保存检查点与数据池状态。这些信息主要用于中断恢复与数据池一致性恢复；而在在线运行阶段，rollback 相关信号主要作为后续查询决策的风险提示，而不是简单地在同一轮内立即回退并重跑。

## IV. 实验设置

### A. 数据集

本文在 Landslide4Sense 基准数据集 [17] 上开展实验。该数据集专门面向滑坡检测研究构建，包含多光谱 Sentinel-2 遥感影像及像素级标注。按照当前实现协议，数据划分如下：

- 初始标注集 $L_0$：151 个样本（3,039 个训练样本中的 5%）
- 未标注池 $U$：2,888 个样本
- 固定测试集 $T$：760 个样本

主动学习总共进行 15 轮训练，其中前 14 个轮次每轮查询 88 个样本，最终标注规模达到 1,383。用于 ALC 归一化的总预算设为 1,519。在更新后的论文结果口径中，我们使用 seed 42 作为代表性个例，并对在 4 个运行中都完整存在的方法（seeds 42-45）进行多 seed 汇总。

### B. 基线方法

本文将 AAL-SD 与以下基线策略进行比较：

1. **Random Sampling**：从未标注池中随机选择样本
2. **Entropy Sampling**：选择平均像素熵最高的样本
3. **Core-Set Sampling**：选择特征空间覆盖度最大的样本
4. **BALD Sampling**：使用 MC Dropout 的 Bayesian Active Learning by Disagreement
5. **DIAL-style**：带不确定性约束的聚类多样性采样
6. **Wang-style**：两阶段 U→K 重排序策略

在更新后的多 seed 分析中，我们进一步区分两个 full-model 变体：`full_model_A_lambda_policy` 作为本文的主 full model；`full_model_B_lambda_agent` 作为允许 agent 显式设置 lambda 的直接对照版本。更早期的 seed-42 单独消融（如 `no_agent`、`fixed_lambda`、`knowledge_only`、query-size 控制版本）仍可用于内部诊断，但由于未在四个 seed 中全部运行，因此不纳入主多 seed 表格。

### C. 实现细节

**分割模型：** 本文采用以 ImageNet 预训练 ResNet-50 为骨干网络的 DeepLabV3+，并在 Landslide4Sense 上进行微调。

**训练参数：** 每轮训练 10 个 epoch，batch size 为 4，学习率设为 $1\times10^{-4}$，并采用常见数据增强策略，如随机翻转、旋转和亮度扰动。

**LLM 配置：** agent 通过运行时配置的 OpenAI-compatible 外部 LLM 接口执行。本文报告实验中的具体模型名称和 temperature 不在训练代码中硬编码，而是由实验环境在运行时提供。

**评估指标：**

- 学习曲线下面积（Area under the Learning Curve, ALC）
- 最终 mIoU
- F1-score

### A. 整体性能对比

表 I 给出了在 Landslide4Sense 数据集 seed 42 设置下各方法的整体性能对比。AAL-SD 对应的 `full_model_A_lambda_policy` 实验取得了 0.6693 的 ALC，同时在所有方法中获得最高最终 mIoU（0.7651）和 F1-score（0.8490）。

| Random | 0.6563 | 0.7391 | 0.8268 |
| Entropy | 0.6674 | 0.7555 | 0.8409 |
| Core-Set | 0.6578 | 0.7507 | 0.8370 |
| BALD | 0.6673 | 0.7644 | 0.8483 |
| DIAL-style | 0.6681 | 0.7626 | 0.8469 |

实验结果表明，混合型方法（DIAL-style、Wang-style 和 AAL-SD）以及不确定性驱动方法（Entropy 和 BALD）整体优于基于多样性的 Core-Set 和随机采样。AAL-SD 取得了最佳最终 mIoU 和 F1-score，说明其自适应控制机制在中后期轮次尤其有效。尽管 Wang-style 的 ALC 略高于 AAL-SD（0.6714 对 0.6693），但其优势主要来自更强的早期轮次收益；相比之下，AAL-SD 在预算耗尽时取得了更优最终性能。这说明当标注池已具备一定覆盖度后，本文提出的 agent 引导策略更擅长在“细化高不确定边界”和“扩展欠覆盖地形区域”之间做出高质量决策。

## VI. 讨论

### A. 方法优势

与传统计算驱动方法相比，将 LLM agent 融入主动学习具有多方面优势。首先，agent 能够同时综合考虑不确定性、多样性、学习进程、预算约束以及梯度导出的风险信号等多种因素，而不局限于单一标量分数。其次，自然语言解释使人工研究者更容易理解并在必要时干预系统决策。第三，自适应权重机制使探索与利用之间的平衡能够随学习阶段变化而动态调整。

AD-KUCS 为不确定性与知识增益的融合提供了一个具有理论直觉且实验有效的统一框架。消融实验也支持这一点：去掉 agent 会同时降低 ALC 与最终精度，而将自适应控制替换为固定 λ 也会削弱最终表现。更重要的是，该框架可以被理解为一种“间接梯度塑形”机制：当查询子集发生改变时，下一轮训练的标注数据分布也随之改变，进而会改变后续优化景观及梯度轨迹。从这一角度看，AAL-SD 不只是一个样本排序策略，而是一个将数据采集与优化稳定性耦合起来的闭环控制系统。

更新后的四 seed 结果进一步强化了这一解释。策略控制版 AAL-SD（`full_model_A_lambda_policy`）并不是每个指标上都最优，但它仍是平均 ALC 最强的方法之一，也是最忠实体现本文“显式闭环策略主导控制”这一理论主张的版本。相对地，agent-lambda 版（`full_model_B_lambda_agent`）取得了最高平均最终 mIoU 和 F1-score，说明在某些条件下，允许 agent 直接控制 lambda 确实有助于终止轮精度。综合来看，策略主导控制更偏向整条学习曲线的效率，而 agent 直接控参更偏向预算末端的最终精度。

### B. 局限性

尽管本文方法表现良好，仍存在以下局限：

1. **计算开销：** 与纯算法式采样相比，LLM agent 推理会引入额外计算成本。不过该成本是在轮次级别摊销的，通常远低于节省下来的人工标注成本。

2. **外部 API 依赖：** 当前实现依赖外部 LLM API，这在数据隐私和可用性方面可能带来风险。未来可探索部署本地小型语言模型。

3. **数据集特异性：** 当前实验主要在单一基准数据集上开展。未来若能在更多滑坡数据集（如 CAS Landslide Dataset）上验证，将更能说明方法的泛化性。

4. **策略敏感性：** 自适应权重策略本身，包括 Sigmoid 先验以及风险感知控制阈值，在不同数据集和场景下可能仍需重新校准。

### C. 实际应用意义

所提出的 AAL-SD 框架对于滑坡检测工作流具有重要实践意义。它能够在维持检测精度的同时减少标注需求，从而显著降低真实场景部署深度学习模型的门槛。此外，方法提供的可解释性对地质与遥感领域专家尤为重要，因为他们往往需要理解并验证自动选样系统的决策依据。

---

## VII. 结论

本文提出了 AAL-SD，一种面向遥感滑坡语义分割的 LLM agent 驱动自适应主动学习框架。其主要贡献包括：(1) 提出带自适应权重的 AD-KUCS 算法，实现探索-利用平衡的闭环调节；(2) 引入 LLM agent 作为可解释决策层，通过可审计工具调用进行上下文感知的候选检查与最终选样；(3) 提出一种基于梯度风险信号调节数据采集的闭环优化视角；(4) 在强不确定性、多样性与混合基线之上进行系统性实验验证。

在 Landslide4Sense 基准上的代表性 seed-42 实验中，策略控制版 AAL-SD full model 获得了 0.6693 的 ALC、0.7651 的最终 mIoU 和 0.8490 的最终 F1-score。更新后的四 seed 分析进一步表明：Wang-style 取得了最高平均 ALC，策略控制版 AAL-SD 以很小差距紧随其后并保持优于多数基线的整条学习曲线效率，而 agent-lambda 版本则获得最高平均最终 mIoU 与 F1-score。这些结果支持一个更克制但更稳健的结论：AAL-SD 的核心价值在于让主动学习决策具备状态感知、可审计和可解释性，而不是简单宣称在固定预算下全面优于所有基线。

未来工作将重点探索：(1) 将该框架扩展至更多遥感任务，如建筑物提取和地表覆盖分类；(2) 探索 LLM agent 的本地化部署方案；(3) 设计更稳健的风险控制机制以支持更长期、更加稳定的主动学习过程。
