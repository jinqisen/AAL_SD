Abstract—Deep-learning-based landslide monitoring has
shown strong potential for automated mapping from remote
sensing imagery, yet its practical deployment remains limited
by the cost of pixel-level annotation. Active learning (AL)
offers a natural remedy by prioritizing informative samples
for labeling; however, existing AL strategies for semantic
segmentation typically rely on a single criterion, such as
uncertainty or diversity, and therefore adapt poorly to changing
learning stages. They also offer limited interpretability for why
specific samples are queried. To address these limitations, this
paper proposes AAL-SD, an LLM-agent-driven adaptive active
learning framework for landslide semantic segmentation. Its
core acquisition module, AD-KUCS, combines uncertainty and
knowledge gain through an adaptive weighting mechanism
governed by a warmup- and risk-aware closed-loop policy.
A Cluster-BalancedK′algorithm corrects the dominant-
cluster capture bias in knowledge-based sampling by applying
saturation-weighted corrections. The LLM agent inspects
candidates in context and finalizes the queried subset with
an interpretable rationale. On Landslide4Sense, a 15-round
evaluation with three independent seeds and seven strategies
shows that the proposed framework achieves competitive
performance (final mIoU 0.703 for the agent-controlled variant)
against strong baselines including Core-Set (0.710) and Entropy
(0.707), while uniquely providing auditable natural language
justifications. An unexpected observation is that lambda
sensitivity remains within 2% across all conditions, suggesting
that under severe class imbalance, the knowledge-uncertainty
balance is a safe rather than sensitive control variable.

approaches, particularly semantic segmentation models such
as U-Net and DeepLabV3+, have achieved remarkable success
in extracting landslide boundaries from optical remote sensing
images [?], [?], [1]. These data-driven methods can automat-
ically learn hierarchical features from raw imagery without
manual feature engineering, substantially outperforming tradi-
tional machine learning approaches.

B. Active Learning as a Solution
Active learning (AL) has emerged as a powerful paradigm
to address the data annotation bottleneck in machine learning.
The core hypothesis of AL is that if learning algorithms can
actively select the most informative samples for annotation,
they can achieve superior performance with fewer labeled
examples compared to passive random sampling. By itera-
tively selecting valuable samples from an unlabeled pool and
querying their labels from an oracle (e.g., human annotators),
AL aims to maximize the learning efficiency under a fixed
annotation budget.
In the context of semantic segmentation for remote sens-
ing, AL becomes particularly valuable because pixel-level
annotation is extremely labor-intensive compared to image-
level classification. The annotation process requires experts to
delineate precise boundaries of landslide areas, which can take
hours per image. Therefore, AL offers substantial potential
to reduce the annotation burden while maintaining detection
accuracy.
Traditional AL strategies for semantic segmentation can
be broadly categorized into three groups: uncertainty-based
methods, diversity-based methods, and hybrid methods.
Uncertainty-based methods (e.g., entropy sampling, margin
sampling) select samples where the model exhibits high
prediction uncertainty, aiming to refine decision boundaries.
Diversity-based methods (e.g., Core-Set) select samples that
maximize coverage of the feature space, promoting explo-
ration. Hybrid methods attempt to combine both perspectives,
often using fixed or manually tuned weights.
Despite progress, existing AL strategies suffer from funda-
mental limitations that hinder their effectiveness in practical
landslide detection scenarios. First, single-metric approaches
cannot dynamically adapt to the learning progress: early-
stage learning may benefit more from exploration, while later
2
stages should focus on exploitation. Second, these methods
lack interpretability; the selection rationale remains opaque
to human operators. Third, most strategies rely on fixed
weighting schemes without adaptive mechanisms to adjust the
exploration-exploitation balance based on model performance
feedback.
C. LLM Agents: A New Paradigm
The recent emergence of Large Language Models (LLMs),
particularly those with Chain-of-Thought (CoT) reasoning ca-
pabilities, has opened new possibilities for intelligent decision-
making in machine learning systems. Unlike traditional al-
gorithmic approaches, LLM Agents can perform complex
reasoning, understand contextual information, and generate
human-interpretable explanations for their decisions. The Re-
Act framework has demonstrated that LLMs can synergize
reasoning and acting by interleaving thought processes with
tool invocations and environmental observations.
Inspired by these advances, we propose integrating LLM
Agents into the active learning decision loop for landslide
semantic segmentation. The agent can receive comprehensive
information about the learning state (e.g., uncertainty scores,
knowledge gain metrics, training progress) and make informed
sample selection decisions through natural language reason-
ing. This integration yields a cognition-assisted, computation-
grounded AL loop, improving contextual decision-making and
interpretability while keeping key control variables governed
by explicit policies in the main full-model setting.
D. Contributions
The main contributions of this paper are fivefold:
1)Novel Framework: We propose AAL-SD, a dual-line
control framework that integrates an LLM agent into
the active learning decision loop via auditable tool calls.
In the main full-model setting,λ tis provided by a
warmup- and risk-aware closed-loop policy, while the

2)Adaptive Algorithm with Cluster-Balanced Sam-

4)Interpretable Decisions: The LLM Agent provides

II. RELATEDWORK
A. Active Learning for Semantic Segmentation
Active learning for semantic segmentation has attracted
significant research attention in recent years. The goal is to
develop query strategies that can effectively identify the most
valuable image patches or superpixels for annotation. Existing
methods can be broadly classified into three categories.
Uncertainty-based Methods.These approaches select sam-
ples where the model predictions are most uncertain. Entropy
sampling computes the average pixel-wise entropy across the
entire image, selecting samples with the highest prediction
entropy. Monte Carlo (MC) Dropout-based methods estimate
uncertainty by measuring the variance of predictions across
multiple forward passes with dropout enabled. Confidence-
based margin sampling selects samples where the difference
between the top two predicted class probabilities is minimal.
Recent work has demonstrated that uncertainty sampling can
achieve comparable performance to full supervision with sig-
nificantly fewer labeled samples in remote sensing segmenta-
tion tasks.
Diversity-based Methods.These approaches aim to select
samples that maximize diversity within the selected batch.
Core-Set selection formulates the problem as maximizing
coverage in the feature space, selecting samples that are most
representative of the unlabeled pool. Density-weighted meth-
ods consider both uncertainty and the density of samples in the
feature space to avoid selecting redundant samples. ViewAL
introduced viewpoint entropy for 3D scene understanding,
showing that incorporating viewpoint diversity improves active
learning performance.
Hybrid Methods.Combining uncertainty and diversity has
shown promising results. DIAL (Deep Interactive and Active
Learning) proposed interactive learning schemes that integrate
user inputs into deep neural networks, along with active learn-
ing strategies to guide users toward the most relevant areas to
3
annotate. The method compared different acquisition functions
including ConfidNet, entropy, and ODIN, demonstrating that
uncertainty-based active learning can quickly lead users toward
model errors. However, these hybrid methods typically use
fixed weighting schemes without adaptive mechanisms.
B. Landslide Detection with Deep Learning
Deep learning has revolutionized landslide detection from
remote sensing imagery. Semantic segmentation models such
as U-Net, SegNet, and DeepLabV3+ have become standard
approaches for pixel-wise landslide mapping [?], [?], [1]. The
DeepLabV3+ model, with its atrous separable convolution and
encoder-decoder structure, has shown particular effectiveness
in capturing multi-scale contextual information for landslide
boundary delineation.
The Landslide4Sense dataset, introduced by Ghorbanzadeh
et al. [2], provides a standardized benchmark for landslide de-
tection research. This dataset contains multi-spectral Sentinel-
2 imagery with pixel-level annotations, enabling fair compar-
ison across different methods. Recent studies have explored
various improvements including attention mechanisms, trans-
fer learning, and multi-modal data fusion to enhance landslide
detection performance.
However, a critical challenge remains: these methods require
substantial labeled data for training. Annotation of landslide
imagery demands expert geological knowledge and significant
manual effort. The class imbalance problem is particularly
severe, as landslide pixels often represent less than 10% of
the total image area. These factors motivate the application
of active learning to reduce annotation requirements while
maintaining detection accuracy.
C. Large Language Models in Machine Learning
The emergence of Large Language Models has transformed
various aspects of machine learning. Models with Chain-of-
Thought (CoT) prompting capabilities demonstrate remarkable
abilities in natural language understanding, reasoning, and
code generation. The ReAct framework demonstrates that
LLMs can synergize reasoning and acting by interleaving
thought processes with tool invocations and environmental
observations. This approach enables agents to maintain sit-
uational awareness, update their understanding based on ob-
servations, and plan future actions.
In the context of data labeling and active learning, prior
work has explored using LLMs as annotators. MEGAnto+
introduced a human-LLM collaborative annotation paradigm
where LLMs create labels that annotators selectively verify
according to active learning strategies. However, these ap-
proaches primarily focus on text or image classification tasks,
with limited exploration in semantic segmentation and remote
sensing applications.

Agents directly into the active learning decision loop for se-
mantic segmentation, enabling dynamic, interpretable sample

A. Problem Formulation
We formalize the active learning problem for landslide
semantic segmentation as follows. LetU={x 1, x2, . . . , x n}
denote the pool of unlabeled remote sensing images,L 0a
small initial labeled set, andTa fixed test set. At each round
t∈ {1, . . . , T max}, a segmentation modelf θis trained on the
current labeled setL t. An acquisition functiona(x, f θ)scores
each unlabeled sample, and the top-kcandidates are queried
for oracle annotation, forming the query setQ t. The labeled
set is updated asL t+1=L t∪Qt, and the process repeats
until the annotation budgetBis exhausted.
A key perspective motivates this work: the query setQ tdoes
not merely add labels but reshapes the empirical training distri-
bution ˆPt+1. Consequently, the optimization objectiveL t+1(θ)
changes, and all subsequent gradient trajectories∇ θLt+1are
indirectly determined by the acquisition decision. This data-
selection-as-distribution-control perspective motivates treating
the acquisition policy as a feedback controller rather than a
static ranking rule.

The AAL-SD framework organizes the pipeline around two

TheActive Learning Linecarries the data flow: the data

The two lines converge at the scoring and agent blocks.
This dual-line design ensures that acquisition decisions are not
driven by a fixed heuristic but are continuously adapted based
on optimization diagnostics, while the agent’s interpretable
reasoning operates within bounds set by the deterministic
control policy.

The adaptive weightλ tfollows a three-phase policy aligned
with the lifecycle of the annotation budget. Each phase corre-
sponds to a distinct operational regime where the relative value
of uncertainty versus coverage shifts based on labeled-set size
and optimization health.

F . LLM Agent Decision System
The LLM agent serves as the interpretable finalization layer
within each active learning round, operating after AD-KUCS
scoring and before the pool update.
1) Agent Architecture:The agent is implemented as a
ReAct loop [5] that interleaves chain-of-thought reasoning
with structured tool calls. At each round, the agent receives:
(1) current learning statistics (round number, mIoU, budget
status,λ t, and risk indicators); (2) the top-kcandidate samples
with theirU(x),K(x), andScore(x)values; and (3) historical
selection patterns. Using this context, the agent performs
multi-step reasoning to inspect candidates, assess spatial and
semantic diversity, and produce a final selection with a natural-
language rationale.
In the primary full-model configuration (Variant A), the
agent does not have access toset_lambda():λ tis de-
termined exclusively by the closed-loop policy described in
Section III-D. This separation ensures that the control logic
remains deterministic and auditable, while the agent focuses on
contextual candidate inspection. A second variant (Variant B)
grants the agent explicitλ-setting authority under constraints,
and is evaluated as an ablation in Section IV.
2) Selection Guardrails:To ensure that agent decisions
remain within operationally safe bounds, we impose hard
constraints applied after each agent selection. Specifically: (1)
the median uncertainty of the selected subset must exceed a
minimum thresholdτ U; (2) the fraction of low-uncertainty
selections is capped at a maximum allowable ratio; and (3)
under high-risk conditions (˜r t> τ high), selections are con-
strained to the high-uncertainty stratum of the candidate pool.
These guardrails define a safe operating region within which
the agent exercises contextual judgment. If a selection violates
any constraint, the system reverts to score-ranked selection for
that round and logs a safety override event, preserving full
auditability.

We compare AAL-SD against six baseline active learning

E. Cluster-BalancedK′Effectiveness
A key contribution of this work is the Cluster-BalancedK′
algorithm (Section III-C3), designed to address the dominant-
cluster capture problem observed in the originalK(x)metric.
BeforeK′: Cluster Capture.Analysis of the original
K(x)-based sampling reveals severe cluster capture bias. In
the feature-space clustering of the unlabeled pool, Cluster 19
alone accounts for over 52% of all labeled samples, while
several minority clusters receive near-zero representation. This
occurs becauseK(x)measures raw distance to the labeled set
without accounting for per-cluster saturation: once a cluster
accumulates many labeled samples, it paradoxically generates
higherdistances (to the nearest labeled neighbor in other
clusters), sustaining a self-reinforcing selection loop.
AfterK′: Balanced Coverage.With the saturation-
weighted correction inK′(x), the sampling from Cluster 19
drops to near 0%, and the annotation budget is redistributed
to previously underrepresented clusters. TheK′correction
effectively orthogonalizes the knowledge-gain direction from
the uncertainty direction, ensuring that the two signals in
the acquisition function provide complementary rather than
redundant information.
Lambda Sweep withK′.To verify thatK′does not
introduce unexpected sensitivity toλ, we conduct a sweep
acrossλ∈ {0.0,0.1,0.2, . . . ,1.0}with the correctedK′(x).The total mIoU range across allλvalues is 1.56%, con-
firming that the flat lambda-sensitivity pattern persists even
after the cluster-capture correction. This result separates two
phenomena: (1)K′successfully fixes the sampling bias in the
knowledge signal, and (2) the limited lambda sensitivity is
an intrinsic property of the task rather than an artifact of the
originalK(x)’s dysfunction.
F . Strategy Stability Analysis
Beyond final accuracy, the reliability of an active learning
strategy across different random initializations is critical for
practical deployment. We measure stability as the range (max-
imum minus minimum) of final mIoU across three seeds.
The stability ranking reveals a clear pattern: simple fixed
strategies achieve the lowest cross-seed variance. Core-Set
leads with a range of only 0.17%, followed by Entropy
(0.71%) and Model B (0.76%). In contrast, complex or adap-
tive methods show wider spreads: Model A (1.99%), BALD
(2.52%), Random (2.95%), and DIAL (3.27%).
This finding carries an important practical implication. For
deployment scenarios where consistency across runs matters
more than marginal accuracy gains, simpler fixed strategies
such as Core-Set or Entropy offer the most reliable perfor-
mance. The higher variance of adaptive methods like Model A
suggests that the policy’s interaction with different random ini-
tializations introduces additional stochasticity, though Model B
mitigates this substantially through the agent’s contextualλ
decisions.

V. DISCUSSION
A. Practical Implications for Remote Sensing Practitioners
The experimental results carry direct implications for prac-
titioners seeking to deploy active learning for landslide seg-
mentation or similar imbalanced remote sensing tasks.
For scenarios where annotation budget is limited and max-
imizing segmentation accuracy is the primary objective, our
results suggest that simple directed strategies offer an excellent
cost-performance trade-off. Core-Set sampling achieves the
highest final mIoU (0.7101) with the lowest cross-seed vari-
ance (0.17%), requiring no hyperparameter tuning beyond the
standard feature extractor. Entropy sampling provides a close
alternative (0.7065) with minimal implementation complexity.
These findings echo recent observations in the broader active
learning literature that sophisticated acquisition functions do
not consistently outperform simpler baselines on real-world
tasks.
However, accuracy alone does not capture the full value
of an active learning system. In operational settings such as
disaster response mapping or geological survey automation,the ability toexplainwhy specific samples were selected for
annotation is essential for building trust with domain experts
and meeting audit requirements. AAL-SD addresses this need
through the LLM agent’s natural language rationale, which
documents the reasoning behind each acquisition decision.
This interpretability comes at a modest accuracy trade-off
(1.3% below Core-Set) but provides a qualitatively different
capability that numerical baselines cannot offer.
B. Lambda Sensitivity: An Unexpected Observation
Interestingly, our experiments reveal that the performance
difference acrossλvalues remains within 2% in all tested
conditions, suggesting that the knowledge-uncertainty balance
may be less critical than commonly assumed for severely
imbalanced segmentation tasks. This observation holds both
before and after theK′correction (lambda sweep range of
1.56% withK′), and the cross-λvariation is comparable to or
smaller than the cross-seed variation for most strategies.
One plausible explanation is that under severe class imbal-
ance (42:1 in Landslide4Sense), the minority-class samples
that are most task-critical occupy a spatially compact and
relatively uniform region in the feature space. Consequently,
both uncertainty-driven and diversity-driven selection converge
on similar subsets of informative samples, regardless of their
relative weighting. In other words, the “task-critical statistics”
of the selected batches may be approximately invariant to the
acquisition criterion when the minority class constrains the
effective search space.
A detailed investigation of this phenomenon, including its
theoretical underpinnings and cross-task validation, is beyond
the scope of this system-oriented paper and will be addressed
in future work. Nevertheless, this observation has a practical
consequence: for practitioners using AD-KUCS, the choice of
λis not a sensitive hyperparameter, and the default three-phase
policy provides a safe, reasonable operating point without
requiring task-specific tuning.
C. Limitations
Several limitations of this work should be acknowledged.
First, all experiments are conducted on a single dataset
(Landslide4Sense) with a specific type of class imbalance.
The generalizability of our findings to other remote sensing
tasks (e.g., urban mapping, crop classification) or different
imbalance ratios remains to be validated.
Second, the LLM agent introduces additional computational
cost and latency. Each round requires multiple API calls
for system status queries, candidate inspection, and selection
finalization. While this overhead is negligible compared to
the annotation cost in practical workflows, it adds complexity
to the deployment pipeline and introduces a dependency on
external API availability.
Third, the lambda sensitivity observation, while consistent
across all our experimental conditions, may not hold for bal-
anced or mildly imbalanced datasets where the diversity signal
plays a more discriminative role. Our results should therefore
be interpreted in the context of severe class imbalance rather
than as a general statement about active learning.
9
Finally, the current evaluation uses a fixed segmentation
architecture (DeepLabV3+ with ResNet-50). The interaction
between the active learning strategy and model architecture
could affect the relative ranking of methods, particularly for
architectures with different feature space geometries.
VI. CONCLUSION
This paper presented AAL-SD, an LLM-agent-driven adap-
tive active learning framework for landslide semantic segmen-
tation from remote sensing imagery. The framework introduces
a dual-line architecture that separates the data acquisition flow
(Active Learning Line) from the optimization diagnostic flow
(Learning Control Line), enabling gradient-derived risk signals
to regulate the acquisition policy in a closed-loop manner.
The core acquisition module, AD-KUCS, fuses uncertainty
and knowledge gain through an adaptive weighting mechanism
governed by a three-phase lambda policy. To address the
dominant-cluster capture problem inherent in the original
knowledge gain metricK(x), we proposed Cluster-Balanced
K′(x), which applies saturation-weighted corrections to redis-
tribute the annotation budget across underrepresented feature
clusters. The LLM agent provides interpretable candidate
inspection and selection finalization with natural language
rationale, offering an auditable decision-making interface for
safety-critical remote sensing applications.
Comprehensive evaluation on the Landslide4Sense bench-
mark with 15 active learning rounds across three indepen-
dent seeds demonstrates that AAL-SD achieves competitive
performance against established baselines including Core-Set,
Entropy, and BALD sampling. While simple fixed strategies
such as Core-Set attain the highest accuracy and stability,
AAL-SD uniquely combines adaptive control, cluster-balanced
sampling, and interpretable agent reasoning in a unified frame-
work. Component ablation confirms the positive contribution
of both the LLM agent and the dynamic weighting mechanism.
An unexpected but consistent observation across all exper-
iments is that lambda sensitivity remains within 2%, compa-
rable to cross-seed variance. This suggests that under severe
class imbalance, the knowledge-uncertainty balance is a safe
rather than sensitive control variable, favoring robust deploy-
ment over fine-tuned optimization.
Future work will pursue three directions: (1) cross-task
validation on balanced and mildly imbalanced datasets to test
the generalizability of the lambda insensitivity observation; (2)
theoretical analysis of the conditions under which acquisition
strategy equivalence emerges; and (3) integration of more ad-
vanced segmentation architectures to evaluate the framework’s
architecture-agnostic potential.
