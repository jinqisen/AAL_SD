## 1. “λ 影响梯度”理论的直接证据

通过对新实验中记录的梯度统计量进行分析，我们现在可以自信地说：**λ 的动态调度确实直接影响了训练过程中的梯度行为。**

### 关键发现：

1.  **Δλ 与 Δ梯度范数呈中等正相关 (r=0.538)**：当 Agent 增加 λ 时，下一轮训练开始时的梯度范数也倾向于增大。这直接证明了 λ 的变化会改变梯度场，从而影响模型的学习起点。

2.  **λ 与梯度衰减率呈中等负相关 (r=-0.551)**：λ 越大（越偏重 Knowledge/Coverage），Round 内的梯度衰减越慢。这说明高 λ 倾向于选择更具多样性但可能更“难啃”的样本，导致模型需要更长时间来收敛。

3.  **λ 与 mIoU 增益呈中等负相关 (r=-0.441)**：这与梯度衰减率的发现一致。高 λ 带来的“难啃”样本虽然长期有益，但在单轮内的 mIoU 增益上可能不如低 λ（偏重 Uncertainty）的样本来得快。

4.  **Train-Val 梯度一致性揭示过拟合**：每个 Round 的训练后期，Train-Val 梯度方向的余弦相似度都大幅下降甚至变为负值。这直接证明了在每个 Round 内部，模型都经历了从“有效学习”到“过拟合”的转变。这为 Agent 动态控制训练 epoch 数提供了理论依据。

**学术价值**：这些直接的梯度证据，将我们对 AAL-SD 的理解从“宏观现象观察”提升到了“微观机理验证”的层面，极大地增强了研究的学术深度和说服力。
