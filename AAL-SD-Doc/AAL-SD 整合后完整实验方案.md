## 3. 实验一：基线对比实验 (P0/P1)

### 3.1 实验目标

横向对比，证明AAL-SD在标注效率（ALC）和最终性能（mIoU）上，显著优于现有主动学习策略，特别是领域内的关键工作。

### 3.2 实验组设置（8组）

| 基线组 | 方法 | 实验目的 | 优先级 |
| :--- | :--- | :--- | :---: |
| B1 | Random Sampling | 证明主动学习的必要性 | P0 |
| B2 | Entropy Sampling | 证明单一不确定性策略的局限性 | P0 |
| B3 | CoreSet | 证明单一多样性策略的局限性 | P0 |
| B4 | BALD | 对比更精细的贝叶斯不确定性度量 | P2 |
| B5 | Fixed λ=0.5 | 证明动态调整λ的必要性 | P0 |
| **B6** | **DIAL-style** | **证明相比领域内关键工作的优越性** | **P1** |
| **B7** | **Wang-style** | **证明相比滑坡检测领域工作的优越性** | **P1** |
| B8 | AAL-SD (Full) | 主方法 | P0 |

## 4. 实验二：细粒度消融实验 (P0/P1)

### 4.1 实验目标

纵向剖析，精确量化AAL-SD每个核心组件的独立贡献，特别是证明**LLM控制器的不可替代性**。

### 4.2 实验组设置（8组）

| 消融组 | 配置 | 实验目的 | 优先级 |
| :--- | :--- | :--- | :---: |
| A0 | AAL-SD (Full) | 基线 | P0 |
| A1 | No LLM Agent（固定λ=0.5） | 量化LLM的贡献 | P0 |
| A2 | Random λ | 证明LLM决策的合理性，而非随机性 | P1 |
| **A3** | **Rule-based Controller** | **最关键消融：证明LLM优于精心设计的规则** | **P0** |
| A4 | No Cold-Start | 量化冷启动策略的贡献 | P1 |
| A5 | Fixed K（全程代表性） | 量化两阶段知识增益设计的贡献 | P1 |
| A6 | BALD Uncertainty | 对比不同不确定性度量的影响 | P2 |
| A7 | No Normalization | 量化得分归一化的作用 | P1 |

### 4.3 关键消融组设计：Rule-based Controller (A3)

为直接回应"为什么需要LLM？"，我们设计三种启发式规则作为对照：
-   **Rule-1（基于性能）**: `if mIoU_increase < threshold: λ_t += 0.1`
-   **Rule-2（基于轮次）**: 早期偏向不确定性（λ小），后期偏向多样性（λ大）
-   **Rule-3（混合）**: 结合Rule-1和Rule-2

**预期结果**：证明即使是精心设计的规则，也无法达到LLM的动态决策质量。

## 5. 实验三：梯度证据分析实验 (P2)

### 5.1 实验目标

构建"LLM调整λ_t → 改变样本分布 → 影响梯度方向 → 导致学习曲线变化"的完整因果链，为AAL-SD的决策提供深层学术解释。

### 5.2 分析方案

1.  **可视化分析**：绘制λ_t轨迹与`train_val_cos`（梯度一致性）的双Y轴折线图。
2.  **相关性分析**：计算λ_t与`train_val_cos`的Pearson相关系数。
3.  **因果链论证**：选择典型的"决策转折点"，深入分析该轮的样本选择、梯度变化、性能变化。
