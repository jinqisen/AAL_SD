摘要

本文在滑坡语义分割主动学习设定下评估 AAL-SD（AD-KUCS + 动态 λ_t 控制）。在固定标注预算与固定每轮训练 epoch 的条件下（N_ROUNDS=15，QUERY_SIZE=88，EPOCHS_PER_ROUND=10），我们比较 full_model 与多类基线，并用 ALC 与最终 mIoU 衡量标注效率。

在 seed=42 的 run 中，full_model 最终 mIoU=0.7665，ALC=0.6696；最优基线（按最终 mIoU）为 BALD（mIoU=0.7685）。

1 引言与研究问题

研究问题 RQ1：在固定标注预算下，AAL-SD（full_model）能否提升标注效率（ALC）与最终精度（mIoU/F1）？

研究问题 RQ2：动态 λ_t（U–K 权衡）是否相对固定 λ 或规则控制带来可量化增益？

研究问题 RQ3：动态 λ_t 的变化是否与训练-验证梯度一致性等“过拟合风险证据”相关联，从而支持风险闭环机制？

研究问题 RQ4：在样本级 U–K 空间，所选样本分布是否呈现与 λ_t 设定一致的偏好？

2 方法概述

AAL-SD 以 AD-KUCS 为基础，对每个候选样本结合不确定性 U(x) 与知识增益 K(x) 进行打分，并通过 λ_t 动态控制二者权衡。

full_model 使用“冷启动→warmup→风险闭环”的 λ_t 日程（见 manifest 中 lambda_policy），旨在早期优先覆盖不确定性、随后逐步引入知识增益并在过拟合风险上升时自适应调整。

3 实验设置

数据与任务：滑坡语义分割（NUM_CLASSES=2），以 mIoU 与 F1 评估分割质量。

预算与轮次：TOTAL_BUDGET=1519，N_ROUNDS=15，每轮标注 QUERY_SIZE=88，初始标注比例 INITIAL_LABELED_SIZE=0.05。

训练口径：FIX_EPOCHS_PER_ROUND=True，EPOCHS_PER_ROUND=10，确保不同方法在训练预算上可比。

硬件与可复现：DEVICE=mps，DETERMINISTIC=True，seed=42。

4 评估指标

主要指标：ALC（学习曲线面积，衡量在整个标注过程中的平均标注效率）。

辅助指标：最终 mIoU / 最终 F1（在最后一轮标注规模下的性能）；曲线稳定性（单步回撤幅度与回撤次数）。

机制证据：train–val 梯度一致性（grad_train_val_cos）与 λ_t 的相关性（Pearson r）。

5 结果与分析

5.1 基线对比（B1–B8）

在基线组（B1–B8）内，full_model 的最终 mIoU 排名为 3/8，ALC 排名为 2/8（排名仅基于本 run 中可用实验组）。

基线组内最优 ALC 方法为 BALD（ALC=0.6702，mIoU=0.7685）。

基线组内最优最终 mIoU 方法为 BALD（mIoU=0.7685，ALC=0.6702）。

full_model 相对传统基线的差值为：Random：ΔmIoU=+0.0103，ΔALC=+0.0124；Entropy：ΔmIoU=+0.0085，ΔALC=+0.0053；Core-set：ΔmIoU=+0.0244，ΔALC=+0.0168；BALD：ΔmIoU=-0.0021，ΔALC=-0.0006。

5.2 消融分析（A0/A1/A2/A3/A4/A5/A7）

消融结果显示：固定 λ（A1）相对 full_model 的 ALC 与最终 mIoU 均下降，支持动态 λ_t 的有效性。

三类 Rule-based 控制（A3）整体弱于 full_model，其中 R2 对 ALC 影响最大，R3 对最终 mIoU 影响最大。

no_normalization 的 ALC 下降幅度最小，但最终 mIoU 仍下降，提示归一化可能主要影响排序稳定性而非最终上界。

5.3 梯度证据（λ_t 与梯度一致性）

full_model 的 λ_t 与 grad_train_val_cos 的 Pearson 相关系数为 -0.538（n≈14 个 round 点）。

该相关性提供了“风险闭环”机制的定量证据：λ_t 的调整与梯度一致性信号存在系统性关系。

5.4 样本级证据（U–K 空间）

在 full_model 的 L3 日志中（round=3），Top-k 与 Selected 的样本数一致（n=88），导致两者 U/K 统计相同；这意味着该轮选择近似等价于直接取 Top-k。

对比 Rule-based(R1)（round=1），Selected 的 U_mean 相对 Top-k 为 +0.1292，显示其更偏向高不确定性样本（与 Fig.5 的分布形态一致）。

5.5 稳定性与异常

为避免仅依赖最终点结论，我们统计了学习曲线的单步回撤幅度与回撤次数（见“异常/稳定性”表）。

需要强调：若某方法出现显著回撤或异常轮次，其最终点可能受随机性或训练不稳定影响，应结合异常报告与稳定性统计进行综合判断。

6 讨论与威胁

  • 当前报告为单 seed（42）结果，未提供统计显著性；建议在 multi-seed 上汇总均值/方差/置信区间。
  • 传统基线中存在自动诊断的“回撤/异常”记录（如 BALD），需要与曲线稳定性统计联合解读，避免仅以最终点做结论。
  • 即便 FIX_EPOCHS_PER_ROUND=true，也要关注训练收敛差异带来的间接影响（例如不同 round 的有效学习强度）。

7 结论

结论：在本次 seed=42 的单 run 中，full_model 在标注效率（ALC=0.6696）与最终性能（mIoU=0.7665，F1=0.8501）上达到强基线水平，并在消融对比中体现出动态 λ_t 与风险闭环组件的贡献。

然而，单 seed 结果不足以给出统计显著性结论；建议以 multi-seed 汇总作为论文主结论依据。

8. 局限与威胁（面向论文写作口径）

  • 当前报告为单 seed（42）结果，未提供统计显著性；建议在 multi-seed 上汇总均值/方差/置信区间。
  • 传统基线中存在自动诊断的“回撤/异常”记录（如 BALD），需要与曲线稳定性统计联合解读，避免仅以最终点做结论。
  • 即便 FIX_EPOCHS_PER_ROUND=true，也要关注训练收敛差异带来的间接影响（例如不同 round 的有效学习强度）。

9. 下一步建议（可直接转为补充实验清单）

  • 跑 multi-seed：复用相同 ablation 矩阵，输出 multi-seed 汇总表（ALC/final mIoU 的均值与 CI）。
  • 对异常基线（如 BALD）补充：复现实验与随机性敏感性分析（seed、dataloader、mps 等）。
  • 把关键结论落到机制：结合 Fig.3/4/5，解释 λ_t 变化、梯度一致性与样本选择分布之间的因果链条。
